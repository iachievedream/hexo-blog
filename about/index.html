以下為自己寫日記與保存整理文字的建議

  • 日記可以使用 Markdown 格式
    它有提供許多段落格式
    以幫助排版

  • 註冊一個 Github 帳號
    就能使用 Github Pages 的免費靜態架站服務

  • 部落格使用 Hexo
    排版簡單
    個人用或部落格分享文章都方便

  • 後續再製作怎樣使用 Github 網站（自我許願）
    與 Git 指令
    完成 Github Pages 的架站過程

這是一個很好用的心智圖工具
