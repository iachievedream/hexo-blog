人類圖

一. 基本概念

  1. 定義：結合占星學、易經、卡巴拉生命之樹與脈輪系統，建立個人能量與決策模式的圖表。
  2. 目標：了解自己的能量類型、決策策略與生命課題，提升自我認知與生活效率。

二. 人類圖的核心組成

1. 類型（Type）

定義個人能量運作模式

類型 英文 描述 全球人口比例（大約）
生產者 Generator 能量運作穩定，主要以回應外界為主 37%
顯示生產者 Manifesting Generator 結合生產者回應與顯示者快速行動能力 33–35%
顯示者 Manifestor 主動啟動行動，帶動他人與環境 9–10%
投射者 Projector 能量不持久，擅長觀察、引導他人 20%
反映者 Reflector 能量受環境影響大，需要觀察與環境同步 1%

2. 策略（Strategy）

每種類型的行動指南

類型 策略 策略說明
生產者 回應（Wait to Respond） 不要主動強迫行動，先感受外界刺激或機會，再依直覺回應
顯示生產者 回應 + 通知（Wait to Respond, then Inform if Needed） 先回應外界訊息（像生產者），再在必要時通知相關人或環境（像顯示者），以減少阻力
顯示者 通知（Inform Before Acting） 可以主動行動，但為了減少阻力，行動前最好通知可能受影響的人
投射者 等待邀請（Wait for the Invitation） 能量不持久，主動行動容易受阻，等待他人的邀請或認可更有效
反映者 月相觀察（Wait a Lunar Cycle） 能量受環境影響大，重要決策最好等一個月亮週期（約28天），以便清楚感受最適合自己的選擇

3. 權威（Authority）

個人內在決策指南

權威類型 描述 注意事項
情感權威 透過情緒起伏做決定 不急於當下，等待情緒平穩再決定
薦骨權威 依靠身體直覺和即時反應 直覺反應即時，但需辨別真實感受
心靈權威 根據意志力或自我需求決定 選擇真正想做的事情
自我投射權威 透過互動與自己說出口的話做決定 從與他人互動中辨別最適合自己的決策
環境權威 觀察外界與他人反應做決定 感受外界環境，選擇最適合自己的方式
月亮權威 反映型專屬，需觀察完整月亮週期做決定 等一個月亮週期後再做重要決策

4. 能量中心（Centers）

對應脈輪概念，影響你的能量流動、思考與行動模式

能量中心 描述 功能
頭腦中心（Head） 靈感與想法來源 提出問題與思考方向
思維中心 / 阿賴耶識（Ajna） 分析與概念化思維 處理資訊、分析問題
喉嚨中心（Throat） 表達與顯化中心 溝通與行動能力
G 中心 自我與方向感中心 決定人生方向與身份認同
心臟/自我中心 意志力與動機來源 決定目標與想完成的事
薦骨中心（Sacral） 身體能量中心 提供持久動力與直覺反應
太陽神經叢中心 情緒中心 處理感受與情緒波動
根部中心 壓力與動力中心 提供行動力與應對挑戰
情緒中心 / 輸入中心 感受環境與他人情緒 辨別哪些感受屬於自己

5. 閘門（Gates）與通道（Channels）

64 個易經卦象對應個人特質

  • 通道：連接兩個中心，形成固定的能量表達模式

6. 輪廓（Profile）

生命角色與人格面向的組合

輪廓 核心角色 核心特質
1/3 調查者 / 冒險者 透過研究與實驗學習，從錯誤中累積經驗
1/4 調查者 / 橋樑者 先調查再建立人脈與影響力，善於連結資源
2/4 隱者 / 橋樑者 擁有自然才華，透過關係被人發掘與引導
2/5 隱者 / 異能者 天生解決問題能力，被他人投射期望
3/5 冒險者 / 異能者 不怕失敗，善於實驗並提供實用方案
3/6 冒險者 / 模範者 生命經驗分三階段成長，從實驗到成熟
4/6 橋樑者 / 模範者 建立人脈影響力，隨時間逐步成為榜樣
4/1 橋樑者 / 調查者 善於建立網絡與研究知識，橋接資源與資訊
5/1 異能者 / 調查者 擅長解決問題並提供實用方案與建議
5/2 異能者 / 隱者 被投射期望，擁有天然能力但需設定界線
6/2 模範者 / 隱者 生命分三階段，最終成為自然榜樣
6/3 模範者 / 冒險者 透過實驗與生活智慧，成為成熟榜樣

7. 定義模式（Definition）

能量中心連接的模式

定義模式 描述 注意事項
單一定義 能量中心彼此連接 行動和決策通常獨立完成
雙重定義 能量分成兩個區塊 需外界或他人「橋接」能量
三重 / 多重定義 能量分成三個或更多區塊 需更多人或環境互動來連接能量
開放 / 無定義 某些中心沒有固定連接 容易受外界影響，需辨別自己的能量

8. 內在課題（Not-Self Theme）

當你沒有遵循自己的策略與權威時，容易出現的典型情緒提示，提醒你偏離自然能量運作

類型 內在課題 說明
生產者（Generator） 挫折感 不等待回應、勉強自己主動行動，容易感到挫折或無力
顯示生產者（Manifesting Generator） 挫折感 + 憤怒 既有生產者回應需求，也有顯示者快速行動特質，未遵循策略可能先感到挫折，再出現憤怒
投射者（Projector） 沮喪 未等待邀請或過度推自己時容易感到沮喪或被忽視
反映者（Reflector） 失望 環境不適合或未跟隨月相節奏行動時容易感到失望或迷失
